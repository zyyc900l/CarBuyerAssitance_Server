# CarBuyerAssitance_Server

本项目是一款基于 Golang Hertz 框架开发的智能汽车购买咨询后端系统，集成 OpenAI API，为用户提供个性化、智能化的汽车选购建议服务。

项目作为《软件工程实践》课程（CodeArts）的核心实践内容，旨在模拟真实企业级开发流程，全面提升团队协作、架构设计、API 开发与工程化落地能力。通过模块化开发、代码规范管理、异步任务处理与持续集成等实践，强化工程思维与团队配合能力，打造一个可扩展、可维护的完整后端服务系统。
启动文件请参考：[文档](./docs/Quick.md)
## 技术栈

- **后端框架**: [Hertz](https://github.com/cloudwego/hertz)
- **数据库**: MySQL
- **消息队列**: Task Queue (用于异步任务，如积分更新)
- **AI接口**: OpenAI API
- **容器化**: Docker
